C++类和对象

发表于 2019-10-31 分类于 C++

类和对象的进一步理解：

  1. 在C++中对象的类型称为类（class）。类代表了某一批对象的共性和特征。
  2. 类是对象的抽象，而对象是类的具体实例(instance)。
  3. 在C++中先声明一个类的类型，然后具体用它去定义若干个同类型的对象。
  4. 对象就是类的这种类型的数据的一个变量。
  5. 类是抽象的，不占用内存，而对象是具体的，占用内存空间。
  6. 对象和类的关系相当于变量和变量类型的关系。
